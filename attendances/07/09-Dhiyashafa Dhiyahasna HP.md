Seminggu ini saya belajar tentang apa saja yang bisa di gunakan dalam statefull widget.
Statefull widget bersifat dinamis, widget ini dapat diperbarui ketika dibutuhkan sesuai dengan action pengguna atau jika ada ada perubahan data. Perubahan data pada statefull widget di trigger oleh perubahan state oleh karena itu sebuah statefull widget selalu memiliki state.
Lalu membuat widget untuk mengkonvensi suhu. Pada saat awal, membuat textformfield agar bisa menginput nilainya. Perbedaannya dengan textfield, textformfield biasa digunakan untuk input pada form yang bersifat dinamis. Juga membuat tampilan keyboard hanya menampilkan angka.
Kita juga membuat event. Selanjutnya adalah membuat event dimana event perhitungan konversi dan update hasil konversi dilakukan saat tombol Konversi ditekan.
Setelah itu, kita memisahkan widget ke dalam 3 widget, yaitu input, result, dan output. Lalu membuat constructor, sebuah widget dapat mengirim paramater ke child widgetnya melalui constructor yang dimiliki oleh widget tersebut parameter ini dapat menyesuaikan dengan kebutuhan yang dimiliki oleh widget tersebut.
